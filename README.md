# Impact Chee

A project for web scraping and machine learning training.

## Project Structure

- `scraper_script.py`: Handles data scraping operations
- `training_script.py`: Handles model training
- `dataset.csv`: Collected dataset for training
- `requirements.txt`: Python dependencies
- `data/pdfs/`: Directory for PDF files

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the scraper:
```bash
python scraper_script.py
```

Run the training:
```bash
python training_script.py
